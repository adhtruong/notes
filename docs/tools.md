## Git

```
git add .
git commit -m 'message'

git commit --amend --no-edit

git rebase <branch>
```

Undo commit

```
git reset HEAD~
```

Undo commit but leave changed files staged

```
git reset --soft HEAD~
```

## Pre-commits
